## [v1.0.0-beta] - 2026-01-19
### Added
- AI insights and agent features (Gemini integration) to provide project recommendations and risk analysis.
- Advanced settings, time-tracker, template library, and analytics dashboard components.
- Initial professional documentation set with multi-page docs and deployment guides.
- Deployment scripts for Google Cloud App Engine and GitHub Pages (`deploy-gcloud.sh`, `deploy-ghpages.sh`).
- Landing page support and CI-friendly build improvements.

### Changed
- Refactor and code-quality improvements: extract constants, improve type safety, and modularize services.
- Build configuration and TypeScript updates for improved compatibility.
- Update landing page links to point to deployed app URL.

### Fixed
- Various bug fixes around AI tool calls, async handling, markdown rendering, calendar date validation, and build regressions.

### Notes
- This is the first beta release of the ODUS v1.0.0-beta offering major AI-driven features and documentation updates.
